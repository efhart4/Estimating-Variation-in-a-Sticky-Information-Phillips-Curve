Format

The focal point of our study is the Phillips Curve incorporating sticky information. The regression equation depicts inflation on the left-hand side, while the right-hand side encompasses the potentially endogenous variable – the output gap \(x_t\), along with a spectrum of inflation expectations and output expectations. To estimate the model, we utilize a set of instruments denoted as \(Z_t\). The required data for model estimation can be summarized as follows: \[\begin{equation*}\pi_t = f(x_t, E_{t-1}\pi_t, .., E_{t-1-j}\pi_t, E_{t-1}\Delta x_t, .., E_{t-1-j}\Delta x_t). \end{equation*}\] and \[\begin{equation*} Z_t = c(1,x_{t-1},x_{t-2},x_{t-3},E_{t-1}\pi_{t+1}) \end{equation*}\]

Inflation Expectation Data

A subset of the independent variables are historical expectations of inflation. Forecasts for the quarterly level of the chain-weighted GDP price index can be used to construct an expectation of inflation using the following formula:

\[\begin{equation*}E_{t-1} \pi_i = log(E_{t-1} P_i) - log(E_{t-1} P_{i-1}). \end{equation*}\] The GDP price index forecasts are the median from the survey of Professional Forecasters. Collected from the Philadelphia Federal Reserve website. This data set runs from 1968-10-01 to 2022-10-01. The file includes 9 columns, two are dedicated to the date, two are annual average forecasts. The other 6 contain our expectation data. For a row labeled time i, we have the expectations using information set i-1. Using this information set, the median forecast is provided starting with \(P_{i-1}\) ending with \(P_{i+4}\). The data is provided below, the date has been formatted and the yearly forecasts have been removed.

The documentation from “https://www.philadelphiafed.org/-/media/frbp/assets/surveys-and-data/survey-of-professional-forecasters/spf-documentation.pdf?la=en&hash=F2D73A2CE0C3EA90E71A363719588D25”. Here for reference.

In the code: Eless5 at time 1970-01-01 is equal to log(E_P$PGDP6[1]) -log(E_P$PGDP5[1]), where E_P$DATE[1] = 1968-10-01. This implies \[\begin{equation*}E_{1968-10-01} \pi_{1970-01-01} = log(E_{1968-10-01} P_{1970-01-01}) - log(E_{1968-10-01} P_{1969-10-01}). \end{equation*}\]

Output Gap Expectation Data

The Real GDP forecasts are the median from the survey of Professional Forecasters, I then assume that the forecasters know the natural rate of output.

The expected change in the output gap can be broken into two pieces given our simplifying assumption. The expectation data will form the first part. While the second part will use the actual change in the output gap. \[\begin{equation*}E_i \alpha \Delta x_t = \alpha*(E_i\Delta y_t - \Delta y^N_t) \end{equation*}\] s.t. \[\begin{equation*}E_i\Delta y_t = log(E_{i}(Y_t))- log(E_{i}(Y_{t-1}))\end{equation*}\] \[\begin{equation*}\Delta y^N_t = log(Y_t^N) -log(Y_{t-1}^N) \end{equation*}\]

E_RGDP = read_excel("Median_RGDP_Level.xlsx");
E_RGDP$DATE <- as.Date(as.yearqtr(paste0(E_RGDP$YEAR, "-", E_RGDP$QUARTER)))
